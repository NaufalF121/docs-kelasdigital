## Pengenalan Google Docs

Google Docs adalah sebuah alat yang digunakan untuk membuat dokumen,
seperti surat, naskah, atau tulisan lainnya. Ini mirip dengan Microsoft Word,
tetapi Google Docs berjalan di web, artinya Anda dapat mengaksesnya
menggunakan website seperti Google Chrome.Selain itu Google Docs juga
tersedia dalam aplikasi seluler untuk Android, iOS, Windows, dan desktop
Google Chrome OS. Melalui Google Docs, pengguna dapat membuat dan
mengedit dokumen secara real-time.

## Manfaat Google Docs untuk Guru Sekolah Dasar

Google Docs memiliki beberapa manfaat yang berguna bagi guru dan siswa
sekolah dasar:
- Mudah Digunakan: Google Docs memiliki tampilan yang sederhana dan
mudah digunakan. Anda tidak perlu menginstal perangkat lunak apa pun,
cukup buka Google Docs di peramban web dan Anda siap untuk
membuat dokumen.
- Akses di Mana Saja: Anda dapat mengakses dokumen Anda dari
perangkat apa pun dengan koneksi internet, seperti komputer di sekolah
atau rumah, laptop, tablet, atau bahkan ponsel. Ini memudahkan akses ke
dokumen Anda kapan saja dan di mana saja.
- Kolaborasi yang Mudah: Salah satu fitur terbaik Google Docs adalah
kemampuannya untuk mengedit dokumen secara bersama-sama dengan
orang lain secara real-time. Ini berarti guru dan siswa dapat bekerja pada
tugas atau proyek yang sama dalam waktu yang sama. Misalnya,
beberapa siswa dapat mengedit dokumen tugas kelompok secara
bersamaan tanpa harus berkumpul di satu tempat.
- Penyimpanan Otomatis: Google Docs menyimpan dokumen Anda secara
otomatis saat Anda mengeditnya. Jadi, Anda tidak perlu khawatir
kehilangan pekerjaan Anda jika terjadi kesalahan atau perangkat mati.
Dokumen juga disimpan di Google Drive, yang merupakan tempat
penyimpanan online, sehingga Anda dapat mengaksesnya kapan saja.
- Pengeditan Teks dan Format: Dalam Google Docs, Anda dapat
menambahkan teks, mengubah ukuran huruf, menambahkan paragraf,
membuat daftar, dan banyak lagi. Anda juga dapat menyisipkan gambar,
membuat tabel, atau menambahkan elemen lain ke dalam dokumen.
- Pembagian dan Komentar: Anda dapat dengan mudah membagikan
dokumen Anda dengan guru atau teman sekelas menggunakan tautan atau
mengundang mereka untuk melihat atau mengedit dokumen. Orang lain

dapat memberikan komentar pada dokumen Anda, memberikan saran,
atau menyoroti perubahan yang perlu Anda lakukan.
- Kemampuan untuk Mengunduh atau Mencetak: Setelah selesai mengedit
dokumen, Anda dapat mengunduhnya dalam berbagai format, seperti
dokumen Microsoft Word atau PDF, atau mencetaknya langsung dari
Google Docs.
