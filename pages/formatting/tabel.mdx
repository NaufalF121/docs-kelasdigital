# Menyisipkan Tabel dan Catatan Kaki
Untuk menyisipkan tabel di Google Docs, Anda dapat mengikuti langkah-langkah
berikut:
1. Klik tempat di mana Anda ingin menambahkan tabel.
2. Klik menu &quot;Sisipkan/insert&quot; di bagian atas layar dan pilih opsi &quot;Tabel&quot;. Di sini
Anda dapat memilih jumlah baris dan kolom yang ingin Anda gunakan.

Hasil tabel 3x3:
[![https://imgur.com/5lniHEu.png](https://imgur.com/5lniHEu.png)](https://imgur.com/5lniHEu.png)

Untuk menyisipkan catatan kaki di Google Docs, Anda dapat mengikuti langkah-
langkah berikut:
1. klik tempat di mana Anda ingin menambahkan catatan kaki
2. klik menu &quot;Sisipkan/insert&quot; di bagian atas layer
3. pilih opsi &quot;Catatan Kaki/footnote&quot; atau dengan menekan shortcut ctrl+alt+F di
keyboard.
[![https://imgur.com/SONvbjv.png](https://imgur.com/SONvbjv.png)](https://imgur.com/SONvbjv.png)
4. Di sini Anda dapat mengetik catatan kaki Anda.
