# Memanfaatkan Fungsi Pintasan
Fungsi pintasan pada Google Docs memungkinkan Anda untuk menghemat waktu
dan meningkatkan produktivitas Anda. Berikut adalah beberapa fungsi pintasan yang
dapat Anda manfaatkan:
1. Pintasan umum: Beberapa pintasan umum yang yang sudah dijelaskan sebelumnya
dapat Anda gunakan pada Google Docs termasuk Ctrl + C untuk menyalin teks, Ctrl +
V untuk menempelkan teks, dan Ctrl + Z untuk membatalkan tindakan terakhir.
2. Pintasan format: Pintasan format memungkinkan Anda untuk dengan cepat
mengubah format teks seperti font, ukuran font, dan warna teks. Beberapa pintasan
format yang dapat Anda gunakan pada Google Docs termasuk:
Ctrl + B untuk menebalkan teks,
[![https://imgur.com/LQILHrx.png](https://imgur.com/LQILHrx.png)](https://imgur.com/LQILHrx.png)
Ctrl + I untuk memiringkan teks,
[![https://imgur.com/9L14uKf.png](https://imgur.com/9L14uKf.png)](https://imgur.com/9L14uKf.png)
Ctrl + U untuk menggarisbawahi teks.
[![https://imgur.com/n3VFWx7.png](https://imgur.com/n3VFWx7.png)](https://imgur.com/n3VFWx7.png)
3. Pintasan navigasi: Pintasan navigasi memungkinkan Anda untuk dengan cepat
berpindah antara bagian dokumen seperti paragraf atau heading. Beberapa pintasan
navigasi yang dapat Anda gunakan pada Google Docs termasuk:
Ctrl + Alt + 1 untuk beralih ke heading level 1
[![https://imgur.com/ugdYrmS.png](https://imgur.com/ugdYrmS.png)](https://imgur.com/ugdYrmS.png)
Ctrl + Alt + 2 untuk beralih ke heading level 2
[![https://imgur.com/zygsbZ5.png](https://imgur.com/zygsbZ5.png)](https://imgur.com/zygsbZ5.png)
4. Pintasan pencarian: Pintasan pencarian memungkinkan Anda untuk dengan cepat
mencari teks dalam dokumen. Beberapa pintasan pencarian yang dapat Anda gunakan
pada Google Docs termasuk:
Ctrl + F untuk membuka kotak pencarian.
[![https://imgur.com/wCoZbpW.png](https://imgur.com/wCoZbpW.png)](https://imgur.com/wCoZbpW.png)
Ctrl + G untuk mencari kata berikutnya.
[![https://imgur.com/askVIBQ.png](https://imgur.com/askVIBQ.png)](https://imgur.com/askVIBQ.png)
5. Pintasan kolaborasi: Pintasan kolaborasi memungkinkan Anda untuk dengan cepat
berkolaborasi dengan orang lain dalam dokumen. Beberapa pintasan kolaborasi yang
dapat Anda gunakan pada Google Docs termasuk Ctrl + Alt + M untuk menambahkan
komentar.
