### Siswa tidak dapat mengirimkan tugas <br />
Jika siswa mendapatkan pesan error "Tidak dapat melampirkan file" saat mengirimkan tugas, coba metode lain berikut:
    - Lampirkan video di Forum, lalu tempelkan link di tugas.
    - Buka folder Tugas Kelas, upload file ke Drive Kelas, lalu bagikan file dengan grup kelas.
    - Upload lagi file yang sama sebagai salinan terpisah, lalu lampirkan file ke kelas.
