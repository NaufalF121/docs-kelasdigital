# Mengimpor dan Menyimpan Dokumen di Format Lain
Anda dapat mengimpor dan menyimpan dokumen Google Docs dalam format lain
seperti Microsoft Word, PDF, atau HTML. Berikut adalah cara-cara untuk
mengimpor dan menyimpan dokumen Google Docs dalam format lain:
1. Mengimpor dokumen: Untuk mengimpor dokumen ke Google Docs dari format
lain seperti Microsoft Word atau PDF, buka Google Drive dan klik tombol
&quot;Baru/new&quot; di bagian atas layar.
[![https://imgur.com/m49c0sj.png](https://imgur.com/m49c0sj.png)](https://imgur.com/m49c0sj.png)
Pilih file upload.
[![https://imgur.com/FdXW1M2.png](https://imgur.com/FdXW1M2.png)](https://imgur.com/FdXW1M2.png)
Pilih dokumen yang ingin Anda unggah dan klik tombol &quot;Buka&quot;. Dokumen akan
diunggah ke Google Drive Anda dan dapat diedit menggunakan Google Docs.
[![https://imgur.com/3AamXnt.png](https://imgur.com/3AamXnt.png)](https://imgur.com/3AamXnt.png)
2. Menyimpan dokumen dalam format lain: Untuk menyimpan dokumen Google
Docs dalam format lain seperti Microsoft Word atau PDF, buka dokumen dan klik
tombol &quot;File&quot; di bagian atas layar. Pilih opsi &quot;Unduh&quot; dan pilih format yang ingin
Anda gunakan. Dokumen akan diunduh ke komputer Anda dalam format yang dipilih.
[![https://imgur.com/JzxAQeq.png](https://imgur.com/JzxAQeq.png)](https://imgur.com/JzxAQeq.png)
