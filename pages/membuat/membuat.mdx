# Membuat Dokumen Baru

Untuk membuat dokumen baru di Google Docs, ikuti langkah-langkah berikut:

1.  Buka browser web dan pergi ke situs Google Docs di alamat docs.google.com.
2.  Jika belum masuk ke akun Google, masuklah menggunakan akun Google Anda.
    Jika belum memiliki akun, Anda dapat membuatnya dengan mengklik &quot;Buat akun&quot; di
    halaman masuk.
3.  Setelah masuk, Anda akan melihat halaman beranda Google Docs.
4.  Klik tombol &quot;Blank&quot; di sudut kiri atas halaman. Ini akan membuka dokumen baru
    dalam tab atau jendela baru.
    <div align = "center">
    [![https://imgur.com/52gYd76.png](https://imgur.com/52gYd76.png)](https://imgur.com/52gYd76.png)
    </div>
5.  Sekarang Anda berada di dokumen baru. Di sini Anda dapat mulai menulis dan
    mengedit dokumen sesuai kebutuhan Anda.
