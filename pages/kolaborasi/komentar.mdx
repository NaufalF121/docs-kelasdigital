#  Menggunakan Pertanyaan Cabang

1. Setelah menambahkan pertanyaan, klik pada ikon "Kotak cabang" (ikon bergambar panah bercabang) di panel pertanyaan tersebut.
2. Setelah mengklik ikon kotak cabang, akan muncul opsi untuk menambahkan kondisi dan aksi untuk cabang tersebut.
3. Pilih pertanyaan sebelumnya yang akan menjadi titik percabangan dalam formulir. Misal, jika Anda memiliki pertanyaan "Apakah Anda memiliki pengalaman dalam bidang ini?", pilih pertanyaan ini sebagai titik percabangan.
4. Selanjutnya, tentukan kondisi dan aksi untuk setiap pilihan jawaban dalam pertanyaan tersebut. Misalnya, jika jawaban "Ya" dipilih, Anda dapat memilih untuk melompati beberapa pertanyaan dan langsung ke bagian berikutnya, sedangkan jika jawaban "Tidak" dipilih, Anda dapat memilih untuk menampilkan pertanyaan tambahan untuk mendapatkan informasi lebih lanjut.

5. Ulangi langkah 3 hingga 7 untuk setiap pertanyaan yang Anda ingin ikut serta dalam cabang.
6. Setelah selesai mengatur pertanyaan cabang, klik tombol "Selesai" di pojok kanan atas untuk menyimpan perubahan.
